API Reference
=============

The 'dqm' package has one class, :class:`DQM <dqm.DQM>`, and several :mod:`utility functions <dqm.utils>`.

Everything can be imported directly from the package. Examples:

.. code:: python3

   from dqm import DQM
   from dqm import plot_frames, get_clusters
